v-if는 조건이 True일 경우에만 엘리먼트를 생성합니다.

v-show는 엘리먼트가 생성되지만 조건이 True일 경우에만 화면상에 출력됩니다.

v-if 조건: count > 10 / 현재 count={{ count }}

v-else-if 조건: count > 5 / count={{ count }}

v-else 조건: 모든 조건 불충족시 기본 출력 / count={{ count }}

v-for 반복 렌더링 예제입니다. (Array)

  • no: {{ index }} / name: {{ item.name }} / tel: {{ item.tel }} / address: {{ item.address }}

v-for 반복 렌더링 예제입니다. (Object)
